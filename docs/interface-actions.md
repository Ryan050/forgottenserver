###### onUse()
> **Description:** This event runs each time a player uses an item by itself or with another item.  
> **Parameters:** <ul><li>_cid_ - _Creature ID that uses the item._</li><li>_item_ - _Use or Use with... (first item)_</li><li>_fromPosition_ - _Starting position_</li><li>_itemEx_ - _Use with... (second item or creature)_</li><li>_toPosition_ - _End position_</li><li>_isHotkey_ - _Boolean, true if hotkey was used_</li></ul>
> **Accepted return values:** Boolean, false for a "You can not use this object." message, otherwise true
