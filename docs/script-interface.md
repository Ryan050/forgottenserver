The scripting interface in The Forgotten Server is supported by the Lua programming language. You can customize the server with scripts for the following interfaces:
* [Action events (using items)](Interface:Actions.md)
* [Chat channel events (joining, speaking and leaving)](Interface:Chat Channel Events.md)
* [Creature events (login, logout and death)](Interface:Creature Events.md)
* [Global events (startup, shutdown, players record, timed, intervals)](Interface:Global Events.md)
* [Monster events (artificial intelligence)](Interface:Monster Events.md)
* [Movements (creature step in/out, moving items)](Interface:Movements.md)
* [NPCs (artificial intelligence)](Interface:NPCs.md)
* [Raids](Interface:Raids.md)
* [Talkactions (speaking in local chat)](Interface:Talkactions.md)
* [Spells](Interface:Spells.md)
* [Weapons](Interface:Weapons.md)

### Metatables
